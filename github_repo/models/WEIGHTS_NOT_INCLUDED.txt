NOTE: Model Weights Not Included
=================================

The actual trained model weights (best_bert_model.pt) are not included in this 
repository due to file size constraints (~440MB for BERT).

To generate the model weights, run:

    python src/bert_classifier.py --data data/sample_labeled_sentences.csv --output results/

This will:
1. Fine-tune BERT on the labeled sentences
2. Save the best model checkpoint to results/best_model.pt
3. Copy the checkpoint to models/best_bert_model.pt

Alternatively, you can use the pre-trained model directly from Hugging Face
and skip fine-tuning for the NLI approach:

    from transformers import AutoModelForSequenceClassification
    model = AutoModelForSequenceClassification.from_pretrained("microsoft/deberta-large-mnli")

The NLI approach (which achieves 76.4% contradiction detection) uses zero-shot
inference and does not require fine-tuning.
